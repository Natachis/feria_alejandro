Evidencias Teóricas y Prácticas del Retoque

🎬 Videos analizados

Análisis de metrajes donde se modifican escenas, se añaden efectos digitales o se altera el audio original para transformar por completo el mensaje inicial.

🖼️ Imágenes modificadas

Casos donde se altera de forma artificial el color, se eliminan imperfecciones de superficies o se estiliza la fisonomía natural de un individuo.

Metodología Técnica: Para realizar esta composición de imágenes, se procesa la cuadrícula de forma automática mediante Dora AI. Posteriormente, se importa el lienzo en Canva donde se distribuyen los elementos y se añaden bocadillos de texto tipo cómic con asistencia de Inteligencia Artificial para extraer fondos limpios. Finalmente, se unifican las tonalidades artísticas globales mediante los filtros cromáticos de CapCut.

💬 Elementos virales (Stickers y Memes)

Expresiones gráficas hiper-condensadas que asocian texto e imágenes con la finalidad de distorsionar eventos cotidianos o esparcir narrativas engañosas en redes sociales.

Flujo de Desarrollo: Inicialmente se aísla el sujeto principal usando motores de segmentación automáticos (Remove.bg o Canva). El elemento limpio se procesa en Canva para la inyección de bordes tipográficos y bocadillos humorísticos. Opcionalmente, se automatizan variaciones estilísticas en Dora AI y se aplica un filtro saturado de alta deformación con el editor fotográfico de CapCut.
